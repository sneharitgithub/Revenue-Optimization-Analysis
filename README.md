Revenue Optimization Analysis

📌 Project Overview

This project analyzes an ecommerce dataset to uncover sales trends, customer behavior, and revenue patterns using SQL.

⸻

🗂 Dataset
	•	Source: Ecommerce dataset
	•	Columns include:
	•	User_ID
	•	Product_ID
	•	Category
	•	Price
	•	Discount
	•	Final_Price
	•	Payment_Method
	•	Purchase_Date

⸻

🧹 Data Cleaning
	•	Converted Purchase_Date from text to date format
	•	Handled null values
	•	Removed duplicates
	•	Standardized column names

⸻

🔍 Exploratory Analysis
	•	Checked distribution of categories
	•	Analyzed pricing and discount patterns
	•	Identified data inconsistencies

⸻

📊 Business Analysis
	•	Sales trend over time (monthly revenue)
	•	Top performing categories
	•	Customer purchase patterns
	•	Payment method contribution

⸻

📈 Key Insights
	•	Certain categories contribute highest revenue
	•	Sales fluctuate across months
	•	Discounts impact final revenue patterns
	•	No strong repeat customer pattern observed

⸻

🛠 Tools Used
	•	SQL (MySQL)
